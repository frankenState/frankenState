## 💼 KEY PROJECTS
### Multilevel Queue Scheduling Simulation

A Java-based simulation of an Operating Systems (OS) CPU Multilevel Queue (MLQ) scheduling algorithm. Built as a university Machine Problem to calculate process execution metrics (Waiting Time, Turnaround Time).


#### 📁 Implementation Approaches

The project showcases two different design patterns for solving the scheduling problem:

* **Static Approach (`src/progs/`)**: Implements scheduling logic linearly using traditional static arrays and parallel structures.
* **OOP Approach (`src/progs2/`)**: A clean refactor using Java Object-Oriented Programming (OOP) and dynamic `ArrayList` collections to manage process states.
